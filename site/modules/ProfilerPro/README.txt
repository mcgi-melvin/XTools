REQUIREMENTS
============

- ProcessWire 3.0.41 or newer is required. 


INSTALLATION
============

1. Copy the files included with the API Explorer ZIP file to this 
   directory (you will need to create the directory):
   
   /site/modules/ProfilerPro/

2. Login to your ProcessWire admin and go to Modules > Refresh. 

3. Now go to Modules > Site. Locate the "ProfilerPro" module in
   the "Profiler" section of modules, and click the "Install" button.
   
The module should now be installed. You can use it by clicking to
the "Setup > Profiler" menu. 

To start Profiling, click the "Config" tab in ProfilerPro, select
the settings you want to use.
